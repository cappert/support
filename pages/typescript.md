## Installation

```bash
npm install -g typescript
```

## Usage

Dans le dossier du projet :

```bash
# génération du fichier tsconfig.json
tsc --init

# compilation
tsc fichier.ts

# compilation avec watcher 
tsc -w fichier.ts
```
